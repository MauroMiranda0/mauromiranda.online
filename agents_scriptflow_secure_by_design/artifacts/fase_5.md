## Diseno de interfaz (F5)

### Direccion visual
- Estilo profesional, tecnico y confiable.
- Paleta sobria con acentos en azul/verde performance.
- Contraste alto en CTAs.

### Tipografia
- Titulares: sans serif moderna de alto impacto.
- Cuerpo: sans serif legible.
- Jerarquia clara (H1 -> H2 -> H3).

### Sistema de color (tokens)
- Primario: azul profundo.
- Secundario: gris grafito.
- Acento: azul tecnologico o verde performance.
- Fondo: claro con secciones alternadas.

### Componentes UI clave
- Boton primario: fondo acento, texto blanco, hover con leve cambio de tono.
- Boton secundario: borde acento, fondo transparente.
- Cards de servicios: icono + titulo + beneficio.
- Cards de proyectos: titulo + descripcion + link.
- Cards de casos: resultado destacado + contexto.
- Barra de navegacion fija con CTA.
- Modal de privacidad con contenido legal.

### Layout
- Grid modular 12 columnas.
- Margenes generosos, aire visual.
- Secciones con divisores sutiles.

### Estados y accesibilidad
- Focus visible en inputs y botones.
- Contraste AA minimo.
- Inputs con labels visibles.

### Seguridad UI
- Formularios con hints claros (que se recolecta y por que).
- Mensajes de error sin detalles tecnicos.
- Links externos con indicador visual.
- Modal con cierre accesible (escape/click fuera).

### Evidencia verificable
- Guia de estilos con tokens.
- Componentes definidos para CTA, cards, formularios.
- Checklist de contraste.

## Recomendaciones para agente 6
- Traducir el sistema visual en componentes implementables.
- Asegurar consistencia en estados hover/focus.
- Mantener la jerarquia visual en mobile.
